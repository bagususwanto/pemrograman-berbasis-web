# Pemrograman Berbasis Web

Repository ini berisi kumpulan tugas-tugas mata kuliah Pemrograman Berbasis Web.

## Daftar Tugas

### [Tugas 1 - SITTA (Sistem Informasi Tracking & Transaksi Ajar)](Tugas.1/)

Implementasi sistem informasi berbasis web untuk tracking dan transaksi bahan ajar dengan fitur:

- Sistem autentikasi (login)
- Dashboard interaktif
- Tracking pengiriman bahan ajar
- Manajemen stok
- Pelaporan dan histori transaksi

Teknologi yang digunakan:

- HTML5
- CSS3
- JavaScript (Vanilla)

[Lihat detail project →](Tugas.1/README.md)
